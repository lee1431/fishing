HOW IT WORKS

밝은 템플릿 페이지는 이렇게 쓰면 됩니다

✓
부드러운 첫 인상

검정 배경보다 가볍고 친근한 인상을 주고 싶을 때 라이트 버전이 더 잘 맞습니다.

✓
정리된 카드 라이브러리

카드와 여백을 중심으로 템플릿을 더 깔끔하게 보여주기에 적합합니다.

✓
브랜드 확장성

향후 업종별 다른 라이트 팔레트로 쉽게 파생 템플릿을 만들 수 있습니다.

✓
상담 연결 유지

색감은 바뀌어도 선택, 상세보기, 문의 흐름은 그대로 유지해 전환 구조를 살립니다.

JSONHUB 테스트 클라이언트

API를 잘 모르는 사람도 바로 테스트할 수 있게 만든 단일 HTML 파일입니다. 프로젝트명과 파일명(name)을 넣고 JSON 저장 / 불러오기 / 목록 조회 / 이미지 업로드를 눌러보면 됩니다.

API BASE:
현재 상태: 확인 전

테스트 입력

저장은 POST /api/json/<project>/<name>
불러오기는 GET /api/json/<project>/<name>
목록은 GET /api/list/<project>

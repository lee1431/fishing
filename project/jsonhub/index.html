1. JSON 데이터 저장 / 불러오기 체험

이 영역의 핵심은 HTML을 수정하는 대신 JSON 데이터를 수정하는 것입니다.
저장은 POST /api/json/<project>/<name>
불러오기는 GET /api/json/<project>/<name>
목록은 GET /api/list/<project>
처음 배우는 사람을 위한 설명
1) JSON 입력창에서 제목이나 설명을 바꿉니다.
2) JSON 저장을 누르면 서버에 저장됩니다.
3) JSON 불러오기를 누르면 서버 데이터를 다시 받아옵니다.
4) 아래 미리보기가 바뀌면, 데이터와 화면이 연결된 것입니다.

          7. 여기서 배우는 핵심 개념
          

            HTML은 구조, JSON은 내용, API는 저장소, JavaScript는 연결입니다.

            아래 데모는 결국 콘텐츠 관리 페이지(CMS)의 축소판입니다.

            글 목록, 상품 목록, 낚시 로그, 갤러리, 포트폴리오가 전부 같은 원리로 움직인다는 점을 체험하는 것이 목적입니다.

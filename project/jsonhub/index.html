JSON 저장 / 불러오기 간단 실습

이 예제의 목적은 단순합니다. 자신이 만든 JSON을 서버에 저장하고, 다시 불러와서 확인하는 것입니다. 아래에서 이미지를 업로드한 뒤, 그 이미지의 전체 URL을 JSON 안에 직접 넣어보면 됩니다.

1. JSON 만들기

저장 주소: POST /api/json/<project>/<name>
불러오기 주소: GET /api/json/<project>/<name>

핵심은 HTML을 수정하는 대신 JSON 내용만 바꾸는 것입니다.

2. 불러온 JSON 미리보기
